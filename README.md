# Jigsaw Puzzle Game

This is a fully client-side HTML puzzle game.

## Features
- Upload your own image
- Choose difficulty (12–96 pieces)
- Smooth dragging & snapping
- Zoom and pan
- Victory animation

## How to Publish on GitHub Pages
1. Upload all files to a public GitHub repository.
2. Go to **Settings → Pages**.
3. Select:
   - Source: **Deploy from branch**
   - Branch: **main**
   - Folder: /**
4. Save — your game will deploy automatically.

Enjoy!
